# Student Score Analyzer

## Description

This project analyzes student marks and provides basic insights such as average, highest, and lowest scores. It also visualizes the data to better understand student performance.

## Dataset

* Source: Self-created (List)
* Description of data: A small dataset containing student names and their corresponding marks.

## Steps Performed

1. Data Cleaning

   * No cleaning required as data was manually created
2. Exploratory Data Analysis

   * Calculated average, maximum, and minimum marks
3. Visualization

   * Bar chart used to compare student performance
4. Model Building (if applicable)

   * Not applicable

## Results

* Identified highest and lowest scoring students
* Calculated average performance of the class

## Tools Used

* Python
* Pandas
* Matplotlib

## Conclusion

The project demonstrates how basic data analysis and visualization can help understand student performance effectively.
